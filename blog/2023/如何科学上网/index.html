如何科学上网

前言

鄙人不才，尝试过很多种科学上网的方式，比如网上各种加速器，有些一开始说免费然后进去就要收费，有些比较良心，每天免费几小时，但是网速真的不忍直视。然后尝试使用v2ray配合网上的免费节点，但是网上的免费节点也不稳定呀，更新订阅更换节点属实累人。本冤种就尝试买了个搬瓦工，照着网上的教程去搭socket（本来是搭v2ray但是自己觉得太慢了），结果不出两天就被墙了。

废话不多说。后面摸索出两条可行路线。一条是clash+一元机场，好处就是机场很便宜，一年十几块的机场费用，然后也不需要自己搭建，简单易用。一条是使用v2ray+境外服务器自建梯子，好处是自己搭建的节点很纯净适合使用gpt等对网络环境要求较高的网站，一个服务器一年也就三百左右，可以和朋友拼，合下来一年也就几十。

正文

clash+一元机场方式

先安装clash软件，这里贴上Android版本和windows版本：

1.clash-for-Android：

GitHub - githubvpn007/Clash-for-Android

2.clash-for-Windows：

GitHub - githubvpn007/Clash-for-Windows

clash软件下载安装好了之后，直接去一元机场官网购买机场，买最便宜的就行，照着里面教程配置即可：

一元机场链接

一元机场注册链接

使用步骤：

  1. 注册账号并购买套餐（推荐最便宜的套餐即可）
  2. 在机场后台复制 Clash 订阅链接
  3. 打开 Clash for Windows，进入 Profiles 标签页
  4. 在顶部输入框中粘贴订阅链接，点击 Download 按钮
  5. 下载成功后，切换到该配置文件
  6. 在 Proxies 标签页选择节点，将出站模式设置为 Rule（规则模式）
  7. 返回 General 标签页，开启 System Proxy（系统代理）

注意事项：

  • 如果无法在软件内更新订阅，可以手动下载配置文件：
    1. 在浏览器中打开订阅链接
    2. 复制全部文本内容
    3. 保存为 .yaml 格式文件
    4. 拖拽到 Clash for Windows 的 Profiles 界面
  • 建议开启 Start with Windows（开机自启）功能
  • 定期更新订阅链接以获取最新节点

v2ray+自建节点

先安装v2ray软件，这里贴上Android版本和Windows版本：

1. v2ray-for-Windows：

v2rayN 使用教程

v2rayN 是 Windows 平台上的 v2ray 客户端，支持多种协议和订阅功能。

2. v2ray-for-Android:

v2rayNG 下载地址

v2rayNG 是 Android 平台上的 v2ray 客户端，功能强大且易于使用。

3. v2fly-docker 部署方式

此方式仅在 Ubuntu 系统上测试过，其他 Linux 发行版应该也可以使用。

步骤：

  1. 下载配置文件
    • docker-compose.yml
    • config.json

  2. 配置节点信息

    编辑 config.json 文件，将以下字段替换为你自己的节点信息：

    • address: 服务器IP地址
    • port: 服务器端口
    • id: VMess 协议的 UUID
    • email: 用户邮箱（可选）

    示例：

    {
  "address": "your.server.ip.address",
  "port": 443,
  "users": [{
    "id": "your-uuid-here",
    "email": "user@example.com"
  }]
}

  3. 部署运行

    将 docker-compose.yml 与 config.json 放在同一个文件夹中，然后执行：

    docker compose up -d

  4. 配置系统代理

    部署成功后，需要在系统代理设置中配置：

    • HTTP 代理：127.0.0.1:7890
    • SOCKS5 代理：127.0.0.1:7891
  5. 停止服务 
    docker compose down

注意事项：

  • 确保已安装 Docker 和 Docker Compose
  • 确保端口 7890 和 7891 未被占用
  • 配置文件路径必须正确，否则容器无法启动

4. 自建节点

自建节点需要购买境外服务器（如 Vultr、DigitalOcean、搬瓦工等），然后在服务器上搭建 v2ray 节点。

推荐服务器提供商：

  • Vultr - 按小时计费，支持多地区
  • DigitalOcean - 稳定可靠
  • 搬瓦工 - 价格相对便宜

搭建教程： 可以参考 v2ray 官方文档 或使用一键安装脚本，如 v2ray-agent。

优势：

  • 节点纯净，适合访问 ChatGPT、Google 等对网络环境要求较高的服务
  • 可以和朋友共享，分摊成本
  • 完全自主控制，不受机场限制

两种方式对比

对比项 Clash + 一元机场 v2ray + 自建节点
成本 约 10-20 元/年 约 300-500 元/年（可多人共享）
配置难度 ⭐ 简单 ⭐⭐⭐ 中等
节点质量 一般，可能被多人使用 纯净，独享
适用场景 日常浏览、视频 访问 ChatGPT、Google 等高质量服务
维护成本 低，机场自动维护 中等，需要自己维护服务器
推荐人群 新手、轻度用户 有技术基础、对网络质量要求高的用户
