# 准备工作

安装相关ros功能包：

-   安装 gmapping 包(用于构建地图):
    ```c++
    sudo apt install ros-noetic-gmapping
    ```
-   安装地图服务包(用于保存与读取地图):
    ```c++
    sudo apt install ros-noetic-map-server
    ```
-   安装 navigation 包(用于定位以及路径规划):
    ```c++
    sudo apt install ros-noetic-navigation
    ```
