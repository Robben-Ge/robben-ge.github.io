# 数据库基础

***

### 数据库简介

&#x20;      数据库，又称为数据管理系统，简而言之可视为电子化的文件
柜——存储电子文件的处所，用户可以对文件中的数据运行新
增、截取、更新、删除等操作。
&#x20;      所谓“数据库”系以一定方式储存在一起、能予多个用户共享、
具有尽可能小的冗余度、与应用程序彼此独立的数据集合。一
个数据库由多个表空间（Tablespace）构成。

### 常见数据库

MYSQL
SQL SERVER
Oracle
Postgres
SQLite

### SQL语法基础

注：SQL语法对于大小写不敏感

#### 查看创建的数据库

```sql
SHOW DATABASES;
```

#### 创建并切换数据库

```sql
CREATE DATABASE test;

USE test;
```

#### 查看数据库里所有表

```sql
show tables;
```

#### 创建表并查看表结构

```sql
CREATE TABLE testtable (
id int(11) auto_increment primary key,
name varchar(256)
);# 创建一个两列的表testtable，id 和 name，id 自动生成


DESC testtable; # 展示 testtable 的表结构

```

#### 添加数据

```sql
INSERT INTO testtable(name) VALUES(‘DASCTF’);
# 往 testtable 表中插入一个数据，id 自动生成，name 为 DASCT
```

#### 查询数据

```sql
SELECT * FROM testtable; #直接查询
SELECT * FROM testtable WHERE name=‘a’; # 查询 name 为 a 的数据
SELECT * FROM testable WHERE name=‘DASCTF’; # 查询 name 为 DASCTF的数据
```

#### 修改数据

```sql
UPDATE testtable SET name=‘DASCTF777’;
# 将表内数据 name 列内容改为 DASCTF777
```

#### 删除数据

```sql
DELETE FROM testtable WHERE name=‘DASCTF777’;
# 删除表中 name 为 DASCTF777 的数据
```
